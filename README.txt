📌 وصف المشروع
───────────────
منصة ويب عربية كاملة لنشر وقراءة القصص والحكايات.
تدعم إضافة القصص وتعديلها وحذفها مع نظام تصنيف وبحث.

⚙️ المتطلبات التقنية
─────────────────────
لا يتطلب المشروع أي تثبيت! يعمل مباشرة في المتصفح.
- متصفح حديث (Chrome, Firefox, Edge, Safari)
- لا يلزم Node.js أو PHP أو أي سيرفر

🚀 كيفية التشغيل (3 طرق)
──────────────────────────

الطريقة 1 — مباشرة (الأسهل):
  افتح مجلد "frontend" وانقر مزدوجًا على: index.html
  سيفتح المشروع مباشرة في متصفحك.

الطريقة 2 — VS Code Live Server:
  1. افتح مجلد "frontend" في VS Code
  2. ثبّت امتداد "Live Server"
  3. انقر بزر اليمين على index.html → "Open with Live Server"

الطريقة 3 — Python (للتطوير):
  1. افتح Terminal في مجلد "frontend"
  2. شغّل الأمر: python -m http.server 8080
  3. افتح: http://localhost:8080
